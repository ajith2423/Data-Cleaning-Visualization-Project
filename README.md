# Data Cleaning & Visualization Project

## Overview
This project focuses on cleaning and visualizing raw Netflix dataset using Python.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Handling missing values
- Removing duplicates
- Data preprocessing
- Data visualization

## Visualizations
- Movies vs TV Shows
- Ratings Distribution
- Release Year Analysis

## Outcome
Learned data cleaning, preprocessing, and visualization techniques using Python.
